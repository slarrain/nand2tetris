tokenizer.py by Santiago Larrain
--------------------
tokenizer.py is a python program that executes project 10

How to run
----------
The code is written in python3 and it uses the following packages:
re
sys
itertools
xml.etree.ElementTree
xml.dom

Usage
-----
The program creates *.xml files from *.jack files with the same name.

$ python3 tokenizer.py inputfile.jack

This programs has not been tested outside a Unix enviroment.

VMtranslator.py by Santiago Larrain
--------------------
VMtranslator.py is a python program that executes project 07

How to run
----------
The code is written in python3 and it uses the 'sys' package and the 'glob' that are
installed by default on python3 installations.

Usage
-----
The program creates *.asm files from *.vm files.

$ python3 VMtranslator.py /path/to/testfile.vm
or
$ python3 VMtranslator.py ../FunctionCalls/StaticsTest/

*********
IMPORTANT:
THE PROGRAM ASSUMES AT LEAST ONE SLASHE ON THE DIRECTORY PATH IF IT IS NOT A FILE
*********
This programs has not been tested outside a Unix enviroment.
